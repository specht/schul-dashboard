# Schul-Dashboard

Das Schul-Dashboard wurde am [Gymnasium Steglitz](https://gymnasiumsteglitz.de/) entwickelt, um SuS und Lehrkräften einen Einstiegspunkt für die täglichen Abläufe im Distanzunterricht zu bieten.
